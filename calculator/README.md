# 绿色建筑工具箱

这是一个在线绿色建筑工具箱，可以用来计算一些绿色建筑评分项指标。

## 功能特点

- 绿色建材应用比例计算
- 计算书一键导出
- 现行绿色建筑评价标准
- 现代化UI界面

## 安装步骤

1. 克隆仓库

```bash
git clone https://github.com/igxy2013/GreenBuilding.git
cd GreenBuilding
```

2. 安装依赖

```bash
pip install -r requirements.txt
```

3. 运行应用

```bash
python app.py
```

4. 访问应用
   打开浏览器访问 http://127.0.0.1:8090

## 技术栈

- Python (Flask)
- TailwindCSS
- HTML/JavaScript
